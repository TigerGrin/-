# ngrok 快速开始

## 🚀 3步完成

### 第1步：启动开发服务器（开发机器）

```bash
npm run dev:h5
```

**保持终端打开！**

### 第2步：启动ngrok（开发机器，新终端）

双击运行：`启动ngrok.bat`

或者命令行：
```bash
ngrok http 5173
```

**复制显示的URL！**（例如：`https://abc123.ngrok.io`）

### 第3步：访问（任何设备）

在其他设备（手机/电脑）的浏览器中：
1. 打开浏览器
2. 输入刚才复制的URL
3. 直接访问！
